# Beer-1

Full-stack brewery discovery and review application for beer enthusiasts.

## What It Does

- browse brewery listings
- view brewery details
- manage favorites
- write and save reviews
- support user accounts and social-style interactions

## Tech Stack

- React frontend
- Node.js and Express backend
- GraphQL and Apollo
- MongoDB with Mongoose

## Project Structure

```text
beer-1/
|- client/   React application
|- server/   API, auth, and data access
```

## Run Locally

```bash
npm install
npm run develop
```

This project is best presented as a portfolio full-stack application demonstrating API design, auth flow, and a consumer-facing UI.
